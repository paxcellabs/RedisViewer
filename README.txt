Summery - Desktop based Redis Viewer to allow multiple connections to REDIS iinstance can be tabbed open.

======================================

Features -

1) Support opening mulitple tabs for different conenctions
2) Store connection settings to load it next time
3) Provides monitor over individual connection to monitor what commands are executing in REDIS instance
4) Proivdes info panel shows redis database information (refreshed every 5 seconds)
5) Only a read only viewer, to view data in redis, enter key pattern to load and list will show up, select any key from list and data will show up with desctipion

======================================

Installation -

Download and run RedisViewer.jar (java -jar RedisViewer.jar) (Executable jar file)
